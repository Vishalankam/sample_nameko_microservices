# sample_nameko_microservices

Example project which helps to build the microservices using nameko

To start the project:
Go to each microservice:
1 - Create the virtual env, and activate it.
2 - Install requirements
3 - Create the config.yaml file(same as above)
4 - Run the service: “nameko run --config ./config.yaml <service_name>"

    - eg - “nameko run --config ./config.yaml adder_service”
